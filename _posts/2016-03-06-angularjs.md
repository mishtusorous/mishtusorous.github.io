Create a HTML file

Add angularjs file to your HTML 
```
<script src="http://ajax.googleapis.com/ajax/libs/angularjs/1.4.8/angular.min.js"></script>
```

Create an application  
```
<div ng-app="">
```

Create a model 
```
<input type="text" ng-model="name">
```

Show it 
```
Hello {{name}}.  
```
